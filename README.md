# Steel-Cutting-Machine
A PLC (Programable Logic Controller) ladder program for Steel Cutting Machine.
Steel Cutting Machine – Mitsubishi PLC Ladder Program

An industrial automation project for a steel cutting machine using Mitsubishi PLC ladder logic with timers, counters, and data registers.

# Features
Automatic cutting cycle
Timer-based operation control
Counter for production tracking
Data register parameter handling
Start/Stop control system
Sequential machine operation

# Technologies Used
Mitsubishi PLC (FX Series)
Ladder Logic Programming
GX Works2

# Future Improvements
HMI integration for parameter control
Fault detection and alarm system
Sensor-based precision cutting
Production data logging
